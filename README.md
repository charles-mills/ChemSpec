The project was built for the Education category of [OpenAI Build Week 2026](https://openai.devpost.com/).

ChemSpec is a chemistry exploration app, giving educators and learners the ability to select arbritary reactions, and watch them play out in a 2D molecular animation taken step-by-step, or in a 3D "human-perspective" view. It includes local, offline support for a massive range of reactions; where it cannot assess the outcome locally, Codex is consulted to build the reaction live.
