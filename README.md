The project was built for the Education category of [OpenAI Build Week 2026](https://openai.devpost.com/).

ChemSpec is a chemistry exploration app, giving educators and learners the ability to see arbritary reactions within the app, in a 2D animation taken step-by-step or in a 3D exterior view. It includes local, offline support for a massive range of reactions; where it cannot assess the outcome locally, Codex is consulted to build the reaction live.
