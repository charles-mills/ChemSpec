# Build Week delivery plan

## Outcome

Ship one polished native ChemSpec journey demonstrating the reviewed lithium
and water outcome from request through evidence, concise `.chems 1`, expanded
structural validation, paired simulation, and explanation.

## Milestone 1 — definitive structural language

Complete the fixed Slices 0–6 in
[`chems-implementation-plan.md`](chems-implementation-plan.md), with every slice
verified and independently reviewed before the next begins.

Exit evidence:

- one normative grammar and specification;
- reviewed lithium/water structures and rule;
- concise source and expanded certificate;
- atom, charge, electron, mapping, and product derivation;
- deterministic structural and observation frames; and
- complete structural conformance report.

## Milestone 2 — provider journey

Implement provider-neutral workflow events and fake-provider integration, then
Codex subscription and direct Responses API providers.

Exit evidence:

- startup selector and accurate preflight states;
- live-search observation research with claim-level evidence;
- strict structured source/evidence output;
- visible bounded repair; and
- explicit provider failure states.

## Milestone 3 — paired native experience

Integrate the validated frame contract into the Iced/wgpu application.

Exit evidence:

- visible workflow, source, evidence, certificate, and derivation;
- distinct covalent, ionic, and metallic rendering;
- synchronized observation and structural playback;
- edit invalidation and stale-result handling; and
- post-playback evidence-linked overview.

## Milestone 4 — release candidate

Harden and rehearse the canonical path.

Exit evidence:

- offline deterministic test suite;
- one opt-in live smoke test per provider;
- formatting, strict lint, docs, conformance, and package gates;
- primary-platform clean-install rehearsal;
- Windows and Linux native build evidence;
- recorded demo and submission copy; and
- persistent representative/explanatory disclosure.

## Cut order

If time becomes constrained, cut in this order:

1. catalogue breadth beyond lithium and water;
2. custom shaders and decorative motion;
3. advanced editor features;
4. secondary-platform installers while retaining build evidence; and
5. supplementary explanation panels.

Never cut deterministic validation, source and evidence visibility, atom
identity, model disclosure, honest Unsupported/Invalid behavior, or the paired
canonical simulation.
