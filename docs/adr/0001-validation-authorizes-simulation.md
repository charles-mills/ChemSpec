# Validation, not reference provenance, authorizes simulation

ChemSpec treats its bundled catalogue as optional reference knowledge and an
optimization, never as an allow-list. Reviewed and provisional derivations keep
distinct factual provenance, but both receive identical renderer capabilities
after crossing the same deterministic kernel validation; missing review or
catalogue coverage cannot by itself block chemistry. This preserves provenance
disclosure without allowing a host pin, review attestation, reaction ID, or
catalogue membership to confer product authority.
