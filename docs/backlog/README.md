# Backlog

This directory contains deferred proposals that are not approved product,
chemistry, or release commitments. A backlog document cannot modify the trusted
catalogue or override the durable contracts in the parent `docs/` directory.

- [`catalogue-content-gap-proposal.md`](catalogue-content-gap-proposal.md) is an
  unattested chemistry-expansion proposal requiring expert review before any
  implementation or trust-root change.
