# Icons

ChemSpec vendors only the SVG icons it uses. Keep each icon set in its own
directory and preserve that set's licence alongside the assets.

## Majesticons

- Source: https://github.com/halfmage/majesticons
- Style: line, 24 x 24
- Licence: MIT (`majesticons/LICENSE`)

Current mappings:

- `atom-2-line.svg`: ChemSpec brand
- `code-block-line.svg`: Codex subscription
- `key-line.svg`: OpenAI API key
- `alert-circle-line.svg`: unavailable-provider warning
- `arrow-right-line.svg`: continue actions
